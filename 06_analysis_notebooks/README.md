# Analysis Notebooks

Jupyter notebooks for interactive exploratory analysis and validation.

## Files

- **polynomial_plot.ipynb** - Polynomial fitting and visualization
  - Interactive curve fitting to datasets
  - Visualization of polynomial degree effects
  - Use for trend analysis and smoothing

- **coordinate_mean.ipynb** - Coordinate averaging and statistics
  - Calculate mean/median of spatial or temporal coordinates
  - Statistical analysis of multi-point datasets
  - Use for averaging multiple measurements

## Running Notebooks

```bash
jupyter notebook polynomial_plot.ipynb
jupyter notebook coordinate_mean.ipynb
```

## Use Case

Interactive exploration of data, validation of analysis methods, and visualization without modifying production code.

## Dependencies

- jupyter
- numpy
- pandas
- matplotlib
- scipy (for polynomial fitting)
